for the module BS2202 Object-Oriented Development.

This project is utilizing Gradle as it's build tool as per in-class demonstrations, therefore before running the project
it will be necessary to build it.

In IntelliJ Idea IDE it can be done by navigating to the Gradle control panel (right hand side of the screen by default)
and accessing BS2202_Summative_V2/Tasks/build/build.

Then it can be run by accessing BS2202_Summative_V2/application/run
